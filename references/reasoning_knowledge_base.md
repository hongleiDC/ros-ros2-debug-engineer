# 推理、公式与代码逻辑知识库

## 目的

项目知识库不仅保存“结论”，还要保存结论如何得到、采用了哪个公式版本、每个数学符号对应哪一个代码变量、单位和坐标系是否一致，以及哪些测试真正验证了这条逻辑。

这套记录用于回答四个问题：

1. 代码实现的数学模型是什么；
2. 每一步推理是否有已验证前提和明确规则；
3. 公式符号、配置参数、消息字段和代码变量是否严格对应；
4. 修改代码后，旧推理、公式、变量映射和测试是否已经失效。
